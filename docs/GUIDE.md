# 导航应用使用指南

## 简介

导航应用是一个现代化的网址导航网站，灵感来自于 [nav.iowen.cn](https://nav.iowen.cn/)。它提供了一个快速、高效的方式来访问您最喜爱的网站。

## 功能介绍

### 🔍 搜索功能

在顶部搜索框中输入关键词，快速查找您要访问的网站。搜索会实时过滤所有分类和链接。

**使用方法：**
1. 在搜索框中输入关键词（如 "GitHub" 或 "视频"）
2. 按 Enter 键或点击搜索按钮
3. 应用会显示匹配的网站

### 🌙 深色模式

点击右上角的设置按钮 ⚙️，勾选"深色模式"选项。

**优势：**
- 保护眼睛，减少蓝光
- 节省电能
- 个性化体验

### 📐 自定义网格

在设置中调整网格列数（2-5列）以适应您的屏幕。

**选项：**
- 2列：适合小屏幕
- 3列：平衡视图
- 4列：默认设置
- 5列：宽屏优化

### 📊 分类浏览

应用包含 8 个主要分类：

1. **🔍 搜索引擎** - Google、百度、Bing 等
2. **👥 社交媒体** - Twitter、Facebook、Instagram 等
3. **💻 开发工具** - GitHub、Stack Overflow、VS Code 等
4. **🎨 设计资源** - Figma、Dribbble、Unsplash 等
5. **📚 学习平台** - Coursera、Udemy、freeCodeCamp 等
6. **🎬 娱乐** - YouTube、Netflix、Steam 等
7. **⚡ 生产力工具** - Notion、Trello、Gmail 等
8. **📰 新闻资讯** - Hacker News、TechCrunch、新浪新闻 等

### 💾 设置保存

所有设置都会自动保存到您的浏览器本地存储中。

**保存的设置包括：**
- 深色模式偏好
- 网格列数设置
- 最近访问的链接

## 键盘快捷键

- `Enter` - 执行搜索
- `Ctrl/Cmd + K` - 获取焦点到搜索框（可选）

## 交互技巧

### 链接卡片

将鼠标悬停在任何链接卡片上会看到：
- 卡片向上移动
- 图标变大并旋转
- 标题变成蓝色
- 阴影加深

### 响应式设计

应用在所有设备上都能完美工作：
- **桌面** (1200px+): 4列默认
- **平板** (768px-1199px): 3列
- **手机** (481px-767px): 2列
- **小屏幕** (<480px): 1列

## 数据隐私

- 所有设置都保存在本地
- 不进行跨域追踪
- 不存储个人信息

## 自定义导航

如果想要添加或修改导航链接，编辑 `src/data/navigation.js` 文件。

### 添加新链接的格式

```javascript
{
  id: 'unique-id',
  name: '网站名称',
  url: 'https://example.com',
  icon: '😀',  // 使用 emoji 表情符号
  description: '网站简短描述'
}
```

## 常见问题

**Q: 如何添加我自己的导航链接？**
A: 编辑 `src/data/navigation.js` 文件并重新构建应用。

**Q: 数据会被保存吗？**
A: 是的，您的设置会自动保存到浏览器本地存储。

**Q: 可以在手机上使用吗？**
A: 完全可以，应用是完全响应式的。

**Q: 如何重置为默认设置？**
A: 在浏览器开发者工具中清除 localStorage，或使用设置中的重置按钮。

**Q: 可以导出/导入导航数据吗？**
A: 是的，可以使用导入/导出功能（如果启用）。
